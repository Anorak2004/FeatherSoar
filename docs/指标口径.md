# 指标口径（P0）

## 会话基础字段
- 运动时长（duration）：结束时间 - 开始时间（秒）
- 卡路里（calories）：实时估算累计（千卡）
- 挥拍次数（strokes）：挥拍检测累计次数
- 杀球次数（smashes）：挥拍检测中标记为杀球的次数
- 正手/反手（forehand/backhand）：挥拍检测中标记的动作次数
- 平均心率（avgHeartRate）：会话内心率均值（bpm）
- 最高/最低心率（maxHeartRate/minHeartRate）：会话内心率极值（bpm）
- 最高拍速（maxSpeed）：会话内拍速最大值（km/h）

## 进攻/防守占比
- 进攻次数 = smashes
- 防守次数 = max(0, strokes - smashes)
- 进攻占比 = 进攻次数 / (进攻次数 + 防守次数)
- 防守占比 = 防守次数 / (进攻次数 + 防守次数)

说明：当前版本进攻/防守仅基于“杀球 vs 其余挥拍”的近似口径；如后续有更细粒度动作分类（如接球/挡球/吊球），可替换口径但需同步更新 Report 图表与说明。

## 趋势数据
- 心率趋势（heartRateSeries）：[{ t: timestamp, v: bpm }]
- 拍速趋势（speedSeries）：[{ t: timestamp, v: km/h }]

说明：当前趋势在 Dashboard 采样后落库，用于 Report/History 绘制折线。
